Esse é um projeto desenvolvido para disciplina Criptomoedas e Blockchain, sem qualquer fim lucrativo.

Modo de uso:

Rode Jocoin_ERC20_Remake

Rode FlashLoanLender, passando o endereço do token como entrada.

Rode Arbitrage passando o endereço do token como entrada.

Transfira Jocoin usando o método Transfer de Jocoin.

Rode FlashLoanBorrower tomando o token, o Lender e Arbitrage como parâmetros.

Agora, é possível realizar o Flash Loan (Caso seja possível pagar as taxas).

Arbitrage garante o lucro de 1% sobre o valor de entrada, enquanto Lender cobra uma taxa de 0.09%.
